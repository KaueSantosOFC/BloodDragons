# Habilidades e Perícias — D&D 5e

---

## Os 6 Atributos

| Atributo | Abreviação | Governa |
|---|---|---|
| Força (Strength) | FOR / STR | Ataques corpo a corpo, carga, atletismo |
| Destreza (Dexterity) | DES / DEX | Ataques à distância, CA, iniciativa, furtividade |
| Constituição (Constitution) | CON | Pontos de vida, concentração, resistência |
| Inteligência (Intelligence) | INT | Magias (Mago), investigação, história |
| Sabedoria (Wisdom) | SAB / WIS | Magias (Clérigo/Druida), percepção, intuição |
| Carisma (Charisma) | CAR / CHA | Magias (Bardo/Feiticeiro/Bruxo), persuasão |

### Modificador de Atributo
```
Modificador = floor((Valor - 10) / 2)
```
| Valor | Modificador | Valor | Modificador |
|---|---|---|---|
| 1 | -5 | 12-13 | +1 |
| 2-3 | -4 | 14-15 | +2 |
| 4-5 | -3 | 16-17 | +3 |
| 6-7 | -2 | 18-19 | +4 |
| 8-9 | -1 | 20-21 | +5 |
| 10-11 | 0 | 22-23 | +6 |

### Geração de Atributos
1. **Rolar:** 4d6, descarta o menor. 6 vezes, distribui.
2. **Compra de Pontos:** 27 pontos. Custo: 8=0, 9=1, 10=2, 11=3, 12=4, 13=5, 14=7, 15=9.
3. **Array Padrão:** 15, 14, 13, 12, 10, 8.

### Teste de Perícia
```
d20 + Modificador de Atributo + Bônus de Proficiência (se proficiente)
```

### Especialização (Expertise)
Dobra o bônus de proficiência para aquela perícia.
- Classes que concedem: Ladino (nível 1 e 6), Bardo (nível 3 e 10).

---

## Vantagem e Desvantagem

- **Vantagem:** Rola 2d20, usa o MAIOR.
- **Desvantagem:** Rola 2d20, usa o MENOR.
- **Ambas ao mesmo tempo:** Se cancelam, rola normalmente (1d20), mesmo que tenha múltiplas fontes de cada.

---

## Testes de Resistência (Saving Throws)

```
d20 + Modificador de Atributo + Bônus de Proficiência (se proficiente)
```

Cada classe concede proficiência em exatamente 2 testes de resistência (definidos na criação).

---

## Percepção Passiva

```
10 + Modificador de Sabedoria + Bônus de Proficiência (se proficiente em Percepção)
```
- +5 se tiver vantagem em Percepção
- -5 se tiver desvantagem em Percepção

---

## Bônus de Proficiência

```
Bônus = ceil(nível / 4) + 1
```

| Nível | Bônus |
|---|---|
| 1-4 | +2 |
| 5-8 | +3 |
| 9-12 | +4 |
| 13-16 | +5 |
| 17-20 | +6 |

## Exploração e Movimento

### Deslocamento por Raça
| Raça | Deslocamento |
|---|---|
| Anão | 7.5m (não reduzido por armadura pesada) |
| Elfo | 9m |
| Elfo da Floresta | 10.5m |
| Halfling | 7.5m |
| Humano | 9m |
| Draconato | 9m |
| Gnomo | 7.5m |
| Meio-Elfo | 9m |
| Meio-Orc | 9m |
| Tiefling | 9m |

### Terreno Difícil
Cada 1.5m de movimento custa 3m (movimento dobrado).

### Saltos
- **Salto em Distância (com corrida):** Distância = valor de Força em pés ÷ 3 (em metros). Sem corrida: metade.
- **Salto em Altura (com corrida):** 0.9m + (mod. Força x 0.3m). Sem corrida: metade.

### Capacidade de Carga
- **Normal:** Força x 7.5 kg
- **Empurrar/Arrastar/Erguer:** Força x 15 kg
