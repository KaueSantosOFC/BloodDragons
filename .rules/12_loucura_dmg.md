# Loucura — Guia do Mestre D&D 5e

---

## Causas
- Magias (*contato extraplanar*, *símbolo*)
- Doenças, venenos, efeitos planares
- Artefatos, visões de horror extremo
- **Resistência:** Teste de SAB ou CAR (ou Sanidade, se usada)

---

## Tipos de Loucura

### Curta Duração (1d10 minutos)
| d100 | Efeito |
|---|---|
| 01–20 | Paralisado (termina com dano) |
| 21–30 | Incapacitado, gritando/rindo/chorando |
| 31–40 | Amedrontado, foge da fonte de medo |
| 41–50 | Balbucia, incapaz de falar ou conjurar |
| 51–60 | Ataca criatura mais próxima a cada turno |
| 61–70 | Alucinações, desvantagem em testes de habilidade |
| 71–75 | Faz o que mandarem (exceto suicídio) |
| 76–80 | Desejo de comer coisas estranhas |
| 81–90 | Atordoado |
| 91–00 | Inconsciente |

### Longa Duração (1d10 × 10 horas)
| d100 | Efeito |
|---|---|
| 01–10 | Compulsão repetitiva |
| 11–20 | Alucinação, desvantagem em testes de habilidade |
| 21–30 | Paranoia, desvantagem SAB/CAR |
| 31–40 | Antipatia a algo |
| 41–45 | Alucinação de estar sob efeito de poção |
| 46–55 | Ligado a "talismã da sorte", desvantagem se distante (+9m) |
| 56–65 | Cego (25%) ou surdo (75%) |
| 66–75 | Tremores, desvantagem FOR/DES |
| 76–85 | Amnésia parcial |
| 86–90 | Ao sofrer dano: SAB CD 15 → efeito de *confusão* por 1min |
| 91–95 | Perde capacidade de falar |
| 96–00 | Inconsciente (não acorda com dano) |

### Permanente (até curada)
| d100 | Defeito |
|---|---|
| 01–15 | "Estar bêbado me mantém são." |
| 16–25 | "Eu guardo tudo que encontro." |
| 26–30 | Imita outra pessoa (roupas, nome, maneirismos) |
| 31–35 | Mente compulsivamente |
| 36–45 | Objetivos acima de tudo |
| 46–50 | Indiferente a tudo |
| 51–55 | Paranoia de julgamento |
| 56–70 | Complexo de superioridade |
| 71–80 | Paranoia de perseguição |
| 81–85 | Confia apenas em amigo imaginário |
| 86–95 | Acha tudo engraçado |
| 96–00 | Prazer em matar |

---

## Cura
| Magia | Cura |
|---|---|
| *Acalmar emoções* | Suprime efeitos temporariamente |
| *Restauração menor* | Cura curta e longa duração |
| *Remover maldição* / *Dissipar o mal* | Pode curar dependendo da fonte |
| *Restauração maior* ou superior | Cura permanente |
